Anthem Journal is the public surface. It contains essays, studies, project pages, and the visible architecture of the practice.

The private archive can sit elsewhere. It may contain working notes, research documents, protocol drafts, private project files, GitHub links, downloads, and implementation material.

A simple access model:

- Public journal: free.
- Private archive: application or payment.
- Consulting: separate fee for setup, adaptation, implementation, or organisational use.

Replace this page with your final access language when the archive is ready.
